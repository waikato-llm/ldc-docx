# ldc-docx
Adds MS Word .docx support to the llm-dataset-converter library.


## Installation

```bash
pip install git+https://github.com/waikato-llm/llm-dataset-converter.git
pip install git+https://github.com/waikato-llm/ldc-docx.git
```
